Which register holds the address of the next instruction?

The Program Counter (PC) holds the address of the next instruction to be fetched in the Fetch-Decode-Execute cycle.

The MDR temporarily holds:

The Memory Data Register (MDR) holds data or instructions moving between memory and the CPU.

Which factor does NOT directly affect CPU performance?

Clock speed, cores, and cache affect performance; USB ports are I/O and don't directly impact CPU speed.
